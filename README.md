# Cooling System

Automatic cooling system based on FastAPI, Flutter and ESP8266. HTTP and WebSocket are used for communication. Data is stored in PostgreSQL and InfluxDB. JWT authentication is used for the user. Device (object and device) identification using custom settings. The system supports the ability to monitor from Grafana and Flutter Application.
